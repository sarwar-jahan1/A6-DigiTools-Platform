# 🚀 DigiTools

## 📌 Description

DigiTools is a modern single-page React application where users can explore premium digital tools, add them to a cart, and manage purchases easily. The UI is clean, responsive, and user-friendly.

---

## ⚙️ Technologies Used

* React.js
* Tailwind CSS & DaisyUI
* JavaScript (ES6+)
* React Icons
* React Toastify
* JSON (for product data)

---

## ✨ Features

* 🛒 Add to cart with real-time cart count in navbar
* 🔔 Toast notifications for add, remove, and checkout actions
* 💳 Dynamic cart system with total price calculation and remove functionality
